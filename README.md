# 🌸 Gerador de Atividades Adaptadas

Aplicativo **100% gratuito e sem chave de API** para professores de educação especial.  
Usa **Pollinations AI** — inteligência artificial aberta, gratuita e sem cadastro.

---

## 🚀 Como colocar no ar (Vercel — grátis)

### Opção A — Upload direto no Vercel (mais fácil)

1. Acesse **https://vercel.com** e crie uma conta grátis (pode usar e-mail ou Google)
2. No painel, clique em **"Add New Project"**
3. Escolha **"Upload"** (não precisa de GitHub)
4. Arraste os arquivos `index.html` e `vercel.json` para a área de upload
5. Clique em **"Deploy"**
6. ✅ Pronto! O Vercel gera um link público como `https://atividades-adaptadas.vercel.app`

---

### Opção B — Via GitHub (recomendado para atualizar depois)

1. Crie conta em **https://github.com**
2. Crie um repositório novo → faça upload dos dois arquivos
3. No Vercel, conecte o repositório e clique em **Deploy**

---

## 📱 Como usar o aplicativo

1. Faça upload da **logo da escola** (opcional)
2. Preencha **escola**, **professor(a)**, **aluno(a)** e **turma**
3. Digite o **tema** (ex: "Frutas", "Animais da Fazenda", "Números 1 a 5")
4. Clique em uma **disciplina**
5. Clique em **✨ Gerar Atividade**
6. Clique em **📥 Baixar PDF**
7. Na nova janela, clique em **🖨️ Imprimir / Salvar PDF**
8. No diálogo de impressão, selecione **"Salvar como PDF"**

---

## ✅ Requisitos

- Conexão com a internet (para gerar via IA)
- Navegador moderno (Chrome, Firefox, Edge, Safari)
- **Sem chave de API · Sem cadastro · Sem custo**
